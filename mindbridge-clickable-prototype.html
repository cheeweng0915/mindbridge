MindBridge Clickable Prototype

This is a clickable front-end prototype for MindBridge. You can press the buttons inside the phone preview to test the flow: Home -> Quiz -> Result -> Resources / Journal / Breathing.

It is still static HTML, CSS, and JavaScript, but the actions are interactive so you can try the experience before building the real Next.js app.

6 screens

Home, Quiz, Result, Resources, Journal, and Breathing.

Real clicks

Buttons inside the prototype actually switch screens and update state.

Sample logic

Quiz scoring, journal saving, and breathing timer all work in the browser.

UI testing

Useful for showing flow, layout, and behavior before the backend exists.

Prototype Flow

How to try it

  1. Start on Home
  2. Click Start Check-In
  3. Answer quiz questions
  4. Submit and view the Result page
  5. Try the Journal or Breathing actions
